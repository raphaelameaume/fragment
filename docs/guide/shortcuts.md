# Shortcuts

| shortcut | action |
|---|---|
|`Cmd/Ctrl + S`| Capture canvas
|`Cmd/Ctrl + Shift + S`| Record canvas
|`r`| Reload sketch
|`w`| Modify layout
|`Space`| Pause rendering
